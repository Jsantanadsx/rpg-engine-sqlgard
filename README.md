# RPG Engine SQLGard

Projeto desenvolvido para simulação de turnos de combate inspirado em RPG.

## Tecnologias utilizadas

- Python
- Flask
- Oracle Database
- SQL / PL-SQL
- HTML
- Vercel

## Como executar

1. Instale as dependências

pip install -r requirements.txt
(flask, oracledb)

2. Execute o projeto via terminal

python app.py

3. Acesse no navegador

http://localhost:5000

## Deploy

Aplicação online:

http://rpg-engine-sqlgard.vercel.app/
